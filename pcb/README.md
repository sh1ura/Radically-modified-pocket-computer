# PCB データ

KiCad でデザインしたPCBのデータを公開します。

![ki2](https://github.com/user-attachments/assets/2601b8d6-2a19-42c4-8434-f2a83632255b)


## 設計手順

1. キーマトリクスの配置を検討（9x10=19本のGPIOで読み取り）
2. 回路図作成
3. キーボードの導電ゴム接点をフットプリントエディタで作成
4. オリジナルの基板をスキャン
5. 画像に合わせてフットプリントを配置
