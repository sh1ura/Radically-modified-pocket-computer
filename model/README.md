# ディスプレイ固定用治具

![スクリーンショット 2025-11-26 23 38 52](https://github.com/user-attachments/assets/9361f9c5-2984-42b2-aad2-1d7ba54f875a)

* ディスプレイの４隅の穴を、M2 のタッピングネジ（とても短いもの）で止めてください。
  - [amazon 等で「M2ネジ 皿頭　タッピング」などと検索して出てくるセット](https://www.amazon.co.jp/dp/B08P1MNV9Z/)の中の、一番短いものを使っています。
* そして、この治具全体を PCB で抑えるようにして止めてください。
